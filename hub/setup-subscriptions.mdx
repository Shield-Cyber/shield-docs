### Add Subscription

To add a new subscription, click the Add Subscription button.  Enter your company name for your initial subscription.  Create additional subscriptions for your clients using that initial parent subscription.

<Frame type="glass">
  <img src="/images/setup-subscriptions-add.png" alt="Setup Subscriptions Add" />
</Frame>

### Edit Subscription

To edit a subscription, click the Edit button in the same row as the subscription.  The Edit button will only be available to users with the Admin permission on the subscription.  The parent subscription and the subscription name can be changed at any time without affecting the data or deployments of the subscription.
